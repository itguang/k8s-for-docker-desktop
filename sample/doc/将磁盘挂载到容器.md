# 卷 -- 将磁盘挂载到容器

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: fortune
spec:
  containers:
    - image: luksa/fortune
      name: html-generater
      volumeMounts:
        - mountPath: /var/htdocs # 把容器 的 /var/htdocs 挂载到 名为 html 的卷
          name: html
    - image: nginx:alpine
      name: web-server
      volumeMounts:
        - mountPath: /usr/share/nginx/html # 把容器的 
          name: html
          readOnly: true
      ports:
        - containerPort: 80
          protocol: TCP
  volumes:
    - name: html # 声明一个名为 html 的单独的 emptyDir 卷,挂载到上面两个容器中
      emptyDir: { }

```

创建两个容器并挂在卷.
```shell script
k create -f fortune-pod.yaml
```

* 临时暴露 Pod 端口
```shell script
k port-forward fortune 8080:80
```
